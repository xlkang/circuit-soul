## Context

Circuit Soul 有三种主题，当前极客主题视觉效果最佳，深色和亮色主题需要优化。

## Goals / Non-Goals

**Goals:**
- 优化深色主题配色和细节
- 优化亮色主题配色和细节
- 添加主题背景效果
- 优化主题切换动画

**Non-Goals:**
- 不改变主题系统架构
- 不添加新的主题

## Decisions

1. **渐变背景**: 深色/亮色主题使用微妙的渐变背景而非纯色

2. **主题色统一**: 三种主题使用统一的主色调体系（极客-绿、暗黑-蓝、亮色-蓝）

3. **平滑过渡**: 主题切换使用 CSS transition 实现平滑过渡

## Migration Plan

1. 更新 CSS 变量定义
2. 添加背景效果
3. 测试三种主题
4. 提交代码
