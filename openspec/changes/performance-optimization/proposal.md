## Why

博客需要遵循 Next.js 最佳实践进行性能优化，提升加载速度和用户体验，同时为 Vercel 部署做准备。

## What Changes

- 使用 next/font 优化字体加载
- 使用 next/image 优化图片
- 添加动态导入动画组件
- 优化首屏加载
- 添加 loading 状态
- 遵循 Vercel React 最佳实践

## Capabilities

### New Capabilities
- `performance-optimization`: 性能优化

### Modified Capabilities
（无）

## Impact

- 影响文件：src/app/*
- 无需新增依赖
