## Context

需要为博客添加联系功能，让访客可以发送消息。

## Goals / Non-Goals

**Goals:**
- 创建美观的联系表单
- 实现客户端表单验证
- 提供成功/错误反馈
- 表单提交使用 mailto 链接（无需后端）

**Non-Goals:**
- 后端邮件服务（后续版本）
- 文件上传
- 实时聊天

## Decisions

1. **mailto 提交**: 使用 mailto 链接提交表单，无需后端服务器

2. **客户端验证**: 使用 HTML5 原生验证 + JavaScript 增强

3. **主题适配**: 表单样式适配三种主题

## Migration Plan

1. 创建联系页面
2. 创建表单组件
3. 添加导航链接
4. 测试提交功能
