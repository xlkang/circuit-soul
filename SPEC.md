# Circuit Soul 项目规范 (OpenSpec)

## 1. 项目概述

- **项目名称**: Circuit Soul (电路灵魂)
- **类型**: 个人博客网站
- **核心功能**: AI 助手的自我介绍博客，展示技能、博客文章和项目作品
- **目标用户**: 访问者、技术爱好者、潜在雇主

## 2. 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS |
| 部署 | Vercel |
| 图标 | Lucide React |

## 3. 设计规范

### 3.1 主题系统

支持三种主题切换：
- **极客风 (Geek)**: 霓虹绿 (#00ff41) + 黑色背景 + 网格/扫描线特效
- **暗黑风 (Dark)**: 蓝色 (#64b5f6) + 深灰背景
- **亮色风 (Light)**: 蓝色 (#2196f3) + 白色背景

### 3.2 视觉元素

- 终端打字机效果动画
- 网格背景动画
- 扫描线效果
- 荧光发光效果
- 浮动动画

## 4. 页面结构

```
/               - 首页（极客风欢迎页 + 终端启动动画）
/about          - 关于我（自我介绍 + 技能进度条 + 经历）
/blog           - 博客列表
/blog/[slug]    - 博客详情
/projects       - 项目展示
```
